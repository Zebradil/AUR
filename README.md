# AUR

AUR (Arch User Repository) packages that I maintain.

## Requirements

Depending on what you want to do, you may need the following tools/packages:

- bash 5
- curl
- diffutils
- docker
- docker-compose
- go-task
- jq
- pacman-contrib
- sed
